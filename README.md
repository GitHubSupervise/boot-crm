# boot-crm
这是自己开发的一个基于SSM框架--简单的客户管理系统，主要功能也比较简单，就是对客户信息进行增删改查
数据库用的是mysql，sql文件已经上传，可用m0001账户，密码为123，进行登录测试，要自己配置一下环境变量等
因为本人主要开发后台，所以前端会很丑，勿喷
